# Ayush Hardeniya's Personal Website

![Website Preview](assets/website-pw-1.png)

This repository contains the source code for my personal website, available at [ayushhardeniya.me](https://ayushhardeniya.me).

## Overview

This project showcases my skills, projects, and journey as a developer and creator. It's built with clean, modern web technologies and serves as my digital presence. The code is open-source for anyone to learn from.

## Features

- **Responsive Design**: Works on all device sizes.
- **Modern UI/UX**: Clean interface with smooth animations.
- **Portfolio Showcase**: Detailed project presentations.
- **Blog**: A collection of "LifeNotes" and "CodeNotes".
- **Reading List**: Books that have shaped my thinking.
- **PWA**: Installable on mobile devices.

## Technology Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Hosting**: GitHub Pages
- **Assets**: Available in [assets](assets) directory
- **Analytics**: Google Analytics

## Getting Started

### Prerequisites

- A modern web browser.
- Basic knowledge of HTML, CSS, and JavaScript for modifications.

### Local Development

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/ayushHardeniya/ayushhardeniya.github.io.git
    cd ayushhardeniya.github.io
    ```

2.  **View Locally**
    Open `index.html` in your browser. For live reloading, you can use an extension like "Live Server" in VS Code.
